Load / Save Object Presets

Use these settings to save a selected object, including child objects, tags, materials, and animations as a preset (these can then be found in the Content Browser under "Presets / User") or, in turn, load such an object.

Even though polygon objects can be saved as presets, we don’t recommend you do this since this can cause the internal preset library to swell enormously.

It’s better to primarily save parametric objects, which actually only consist of stored object settings.

Drag & drop an be used to place objects into the Content Browser’s Presets / User folder or from this folder into the Viewport or the Object Manager.
