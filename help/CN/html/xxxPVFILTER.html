Color Correction

This post effect lets you make common color corrections right away without having to switch to a different application. Color correction in the form of a post effect will affect the (internal) rendering process in full color depth (32-bit). Differences in color correction can occur if it is used as a post effect or filter (see below). The latter may already be used with reduced color depth.

The color correction Filter can also be used. It can be enabled in the Picture Viewer and its parameters can be converted directly to a post effect (Post Effect button).
