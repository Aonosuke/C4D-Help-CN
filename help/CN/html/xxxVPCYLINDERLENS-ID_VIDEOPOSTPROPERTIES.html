Cylindrical Lens

Basic Properties

Horizontal Field of View [1..360°]

Use this setting to define how much of the horizontal view should be taken into account for calculation. A value of up to 360° can be entered.

Vertical Size [0.01..1000000m]

Use this setting to define the size of the parallel camera. Higher values will enlarge the portion of the image shown and will compress the image.
