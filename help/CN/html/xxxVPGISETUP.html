GI Setup

First some basics regarding parameters.

The standard GI settings described herein have been set up and defined using presets to offer the best, error-free results for most applications. Basic settings are located in the General tab.

Don’t worry about the other GI parameters until/unless you want to delve deeper into GI - fine tuning, reducing render times, etc.
