开始

要开始使用 CINEMA 4D 或者 BodyPaint 3D，你可以：

  • 双击程序图标。
  • 双击一个场景文件。
  • 使用开始菜单（Windows）。
  • 你还可以从 Explorer (Windows) 或者 Finder（Mac OS）拖放一个或多个文件到应用程序图标上，或者直接拖进程序中。
  • 通过命令行工具。
  • CINEMA 4D Lite (也是另一个 CINEMA 4D 版本) 是直接从 Adobe After Effects 中运行的。

快捷键 1 到 8

要使用快捷键，按下键并拖动鼠标。视图面板的默认快捷键是：

1 移动摄像机

2 沿视图方向移动摄像机

3 旋转摄像机

4 移动对象

5 缩放对象

6 旋转对象

7 移动对象（不包含子对象）

8 套索选择

用户界面

 

CINEMA 4D 用户界面有很多 Windows 或者 Mac OS 所没有的特性：

  • 你可以把所有窗口停靠在主窗口中。
  • 当你移动一个停靠的窗口，它周围的窗口会自动适应尺寸。
  • 你可以将窗口显示为标签来节省显示空间。

CINEMA 4D 用户界面可以自由地配置，你可以创建你自己的图标调板（包括折叠的图标）、编辑菜单。你可以自定义各种布局并且在它们之间自由地切换。例如，分别创建一个建模界面、一个贴图界面和一个动画场景界面是很有用的，因为这些任务会用到不同的管理器面板。 另一个强大的特点是，功能可以被集成到用户界面中并利用其特性。

切换界面最快的方法是点击窗口右上方的界面按钮，在弹出的界面和菜单列表中选择你想要的界面，你可以将你自己的界面添加到这个列表中。

要切换到其他预设的界面，点击窗口右上方的界面按钮，并选择你想要的界面。

快速导航

这个部分会在你第一次启动 CINEMA 4D 时显示并引导你浏览用户界面。

主菜单

 

主菜单包含了最重要的 CINEMA 4D 命令和功能。任何额外安装的程序版本（如 CINEMA 4D Studio 等）或者添加的功能的菜单也会在这里显示。

管理器

管理器是 CINEMA 4D 中的主程序要素，每一个管理器都有独立的窗口并与其他管理器一起运行，这意味着每个管理器可以独立地操作。这让你可以在图片查看器中渲染动画的同时在视图面板中进行操作。

窗口

每个管理器都有它自己的窗口，你可以自由地把它们停靠在 CINEMA 4D 的主窗口中。在默认界面布置中，最重要的功能已经停靠在主窗口中了。如果你改变了停靠的窗口的尺寸，它周围的窗口也会自动更改尺寸。

要解除停靠一个窗口，可以在它的手柄图标上点击并选择解锁。要重新停靠解锁的窗口，则可以拖动它的手柄图标放到主窗口上，一个亮色条会显示它插入的位置。要改变一个窗口为尺寸，先把鼠标移动到它的边界，光标会变成一个表示方向的双向箭头（水平或者垂直），这时你就可以根据需要拖动边界了。窗口组中的其他窗口也会适应地变化尺寸。

菜单栏

如果没有足够的空间显示全部菜单栏，则会显示一个黑色的三角箭头，点击这个箭头来会显示剩下的菜单。

图标调板

默认的界面有两个包含大部分常用命令图标的图标调板（也就是工具栏）。一部分图标有一个小黑色三角，表示包含有折叠的命令组。要使用命令组，需要在图标上点击并按住鼠标左键。注意，展开群组之前显示的图标是最近常用的命令（前提是锁定图标在折叠群组的上下文菜单中被禁用）。

这里是关于图标调板创建的描述。

标签

你可以标签化显示窗口和图标调板。例如在默认界面中，对象管理器和结构管理器是作为标签显示的。这节省了显示空间，并且可以让你快速找到管理器或者图标。如果标签没有足够的空间全部显示，会出现一个小滚动按钮，你可以用它滚动到隐藏的标签，你可以点击窗口右上角的小三角图标或者拖动其中一个标签来滚动整个标签。

上下文菜单

可以通过单击鼠标右键打开上下文菜单。

视图面板

视图面板是一组最多四个视窗的组合，它也是程序的核心，是你建立 3D 模型和动画的地方。

对象管理器

对象管理器包含场景中所有的元素（对象）。你可以层级化地群组对象。例如你可以把一片树叶作为一根树枝的子对象，而这根树枝是一棵树的子对象。如果你想要编辑一个对象，需要先在对象管理器中点击它的名字或图标来选择它，对象的名字会变成黄色表示被选中。大部分对象拥有可以添加诸如材质或平滑着色特征的标签。

编辑器视图中的物件可以利用 CINEMA 4D 强大的图层功能来隐藏或显示。

材质管理器

材质管理器包含场景中所有的材质。如果你双击一个材质图标，材质编辑器会被打开，然后你可以对材质进行编辑。你可以通过从材质管理器中拖放材质球到对象上来为其添加材质。

坐标管理器

你可以使用坐标管理器来进行精确的建模和操作。例如，比起在视图面板中缩放一个对象，你可以在坐标管理器中输入准确的数值。跟所有的数值文本框一样，你可以输入数学运算符号。

动画工具栏

对象的位置、比例、旋转或其他性质可以通过记录按钮来进行记录。动画调板是一个时间线的简化版，可以用来创建更简单的动画而不需要切换到动画界面。这里是功能更多的时间线的描述。

属性管理器

属性管理器显示了被选择对象的所有特性和设置：材质、着色(shader)、标记节点(tag node)、跟踪（tracks)和关键帧(keys)等。这些特性和设置也可以在属性管理器中进行编辑。

许多 CINEMA 4D 工具的特性也会在这里显示。

建模方法

可以用多种方法来创建 3D 对象，当然这些建模方法也可以在 CINEMA 4D 中使用，其中最主要的如下：

构造几何实体

这是所有建模方式中最简单的一种，所有的对象由圆柱体、球体、圆锥体等已有的参数化初始几何体构成。

  • 主菜单：创建 / 对象（链接）。
    这些对象非常适合用来制作有无限多可能性的克隆对象。
  • 主菜单：运动图形（链接）。

多边形建模

这是一种经典的建模方法，可以使用各种工具来创建、合并和编辑多边形。虽然比较枯燥，但这种方法可以让你完全控制最终创造的对象。细分曲面（SDS，Subdivision Surfaces）是基于多边形建模的。

  • 主菜单：网格 (链接)

雕刻

雕刻可以创建非参数模型，类似于使用特殊工具来添加或削减材料的真实雕刻。 (造型刷).

  • 在 CINEMA 4D 界面的右上角选择Sculpting界面。

生成器

生成器可以与样条结合使用来帮助创建对象。样条让你可以创建各种类型的封盖（caps）和表面。虽然这是一种非常简洁地建模方式，但是它不会用在角色建模（modeling characters）上。

  • 主菜单：创建 / 生成器（参考这里或者这个链接）。

细分曲面

这是目前最先进的建模方法。包含数个多边形的对象被动态细分,以创建平滑的、有机的高分辨率对象。通过使用边加权（edge weighting）还能产生清晰的边缘。当使用细分曲面工作时，你应该对多边形建模有很好的理解，因为被细分的对象会使用多边形编辑。（见上文）

  • 主菜单：创建 / 生成器 / 细分曲面（链接）

融球

诸如流体（与粒子结合）这样的有机造型可以用融球结合球体、样条、点来创建。

  • 主菜单：创建 / 模型 / 融球（链接）

布尔 / 变形器

这些修改器（modifiers）可以对通过上述方法创建的对象进行变形，以此创建新的对象。

  • 主菜单：创建 / 造型 / 布尔（链接）或者：创建 / 变形器（链接）。

选择哪种建模方法取决于要创建的对象。你需要知道的是：任何物体都可以建立模型，用上述的方法没有不能创建的东西。

注意，对象管理器的层级允许不同建模方法相结合。例如，你可以使用各种变形器来对低分辨率对象（如一个立方体）进行变形，把它放到一个层级中，对它使用变形器，然后使用布尔将它切掉一块。
