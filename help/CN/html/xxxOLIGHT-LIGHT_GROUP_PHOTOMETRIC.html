光度

概述

各种 IES灯光（方向如图中左边所示）照射到墙壁和地面上。

使用 IES（Illuminating Engineering Society；扩展名：*.ies）格式的灯光可以让光源表现得像真实灯光一样。如果你仔细查看上图，会发现灯光的光线散步十分复杂。这些主要是利用特殊的反射器产生的，反射器会集中、分散或用其他方式传播光线（取决于使用的灯光类型）。

IES 文件包含给定光源周围光线的空间散布（以 ASCII 文件的形式）。这些文件是在实验室测试中生成的（因此也被称为“光度灯光”），并由灯具生产商在线上发布。

在 CINEMA 4D 中，IES 文件可以在图片查看器或内容浏览器中显示：

左边是显示在图片查看器/内容浏览器中的 IES 文件。光源的 Z 轴指向下方。右边是一个常见的 IES 文件图示，但是理解上会更加困难，通常你会在灯具产品目录中看到这种说明图。

使用内容浏览器你可以方便快速地直接将整个 IES 目录添加到场景中，或将单个 IES 文件拖放到灯光的文件名文本框中。

一个德国欧科照明有限公司的光源。

一个 IES 光源就是一个带有全部 CINEMA 4D 参数的灯光。这意味着你可以给它添加如镜头光晕或噪波等效果。

如何创建一个 IES 灯光？

  • 新建一个灯光
  • 在常规标签页，将类型设为IES
  • 移动和旋转灯光，让它的 Z 轴指向发射光线的方向
  • 在光度标签页，加载一个 IEF 文件到文件名文本框中

光度强度

如果不启用，会在光源原点处使用带有平方倒数衰减的极亮光源，这经常会让 IES 灯光过亮。不过优点是它是独立于 CINEMA 4D 的单位的（无论你的模型单位使用的是毫米还是公里）。

这也意味着如果你的模型是物理准确/真实比例的，你应该启用这个选项（通常启用这个选项能让结果物理上更准确）。

注意：这个选项不是对所有 IES 文件都适用（取决于它们的内部结构）。

强度 [0..10000000]

This intensity in Candela is the maximum intensity of the IES. The remaining brightness values will be applied in relation to this parameter and can then be adjusted in correlation with it.

单位

In addition to 强度, these 3 units can be applied to influence the light’s intensity (and they are not at all restricted to use with only IES lights):

  • Candela (cd): The light intensity is only dependent on the 强度 value, independent of light shape (e.g., Spot) or size.
  • Lumen (lm): The light intensity is primarily dependent on the light shape, i.e., a Spot light’s intensity, for example, increases when its cone is narrowed.
  • Lux: If the 光度强度 option is enabled (Area light), the light intensity can be adjusted in Lux. The light intensity is, logically enough, dependent on the size of the light source (e.g., Size X). If, for example, the light’s surface is doubled, its intensity will also be twice as strong.

光度数据

Enables or disables the evaluation of IES data. If disabled, the light source will revert to being an Omni light source with adjustable Lumen or Candela intensity.

文件名

You can drag an IES file into this field where its name and complete path (location) will be displayed.

Attention!: The IES file’s path will be displayed here but the IES data will be saved within the Cinema 4D file. Hence, the IES file itself does NOT have to accompany the Cinema 4D file when transferring files.

光度尺寸

Very often IES files contain information regarding the size of the actual light. Enabling this option will turn the IES light into an Area light (with the usual settings in the Details tab, which will be assumed from the IES file).

Otherwise the IES light will function as an Omni light.

 Information

Here you will find various information saved within the IES file. However, not all IES files will contain this information. These fields will then remain empty.
