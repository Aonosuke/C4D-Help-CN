Function available in CINEMA 4D Visualize, Broadcast, Studio

Share Machine over Network

Use this command to remove or add the current computer to Team Render (can also be done while Team Render is in use).

If the current computer is removed during rendering, the process will be continued when the computer is added again using this command (if the other render clients have not yet ended the process in the meantime).

Hence, this command is useful if you want to temporarily free up memory on the computer on which you’re currently working.
