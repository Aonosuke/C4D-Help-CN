Manual Reference CINEMA 4D CINEMA 4D Prime Views and Viewports
Function available in CINEMA 4D Prime, Visualize, Broadcast, Studio & BodyPaint 3D

Panel

Each view panel can have up to four view panels. Each of these viewports may have its own:

  • camera
  • projection type
  • display mode

Panels

You can choose a single-view mode or all-views mode. Choose from:

Toggle Active View

When in multiple Viewport display mode, this function toggles the active view between full-screen and normal size. This can be done faster by clicking the center mouse button.
