Standard values for the current scene can be defined in the Project Settings. These are settings that affect the scene globally and are not defined by objects, render settings, etc.

The Project Settings can be opened in the Attribute Manager using one of two methods:

  • Main menu: Edit | Project Settings
  • Attribute Manager: Mode | Project
