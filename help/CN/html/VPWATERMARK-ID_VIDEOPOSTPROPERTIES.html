启用或禁用显示文本框。

动态界面

若启用，文本框条目将被排序，因此将使用最少的列数（否则顺序将直接反映对话框窗口选项的顺序）。

显示标签

时间码
渲染时间

日期
工程名称
面数

机器信息

RELEASE 17

场次

若启用，会显示相应的信息。

页脚

若启用，你可以输入自己的页脚文本。如果文本太长，它会在图像的右边缘断行。

文本

在这里输入你的页脚文本。文本被限制为一行，并且在其过长的情况下断行。

可以使用以下变量（也可以点击文本框右侧的按钮来使用），它们之后会变成相应的信息：

通用信息

$c4dtype：目前只在 64 位下可用。

$c4dver：CINEMA 4D 版本。如：“14.014”。

$buildid：CINEMA 4D build 版本，它会在程序启动时显示。

$ostype：使用的操作系统。如：“Windows 64-bit”、“OS X 64-Bit”等。

$osver：操作系统版本。如：“Windows XP Professional Service Pack 3 (build2600)”、“OS X 10.6.6”等。

$cputype：CPU 类型。如：“GenuineIntel, stepping 11, model 15, instruction family 6”或“simply Intel”等。

$cpuname：CPU 名称。如：“GenuineIntel, stepping 11, model 15, instruction family 6”或“Intel Core i7 CPU 860”等。

$cpughz：CPU 频率。如“2.8 (Gigaahertz)”。

$cpucnt：CPU 核心数量。

$cpuhtcnt：每个 CPU 或核心的超线程数量。

$machine：输出的内容与机器信息选项相同。

$compname：计算机或网络名称。

RELEASE 17

$trcompname：设置 | 渲染器 | Team Render 菜单的电脑名称中设置的 Team Render 客户端名称。

$username：当前登录的用户名称。

$doccopyright：工程设置的信息标签中设置的版权。

$docauthor：工程设置的信息标签中设置的作者。

$file：渲染的 CINEMA 4D 场景名称。

$path：渲染的 CINEMA 4D 场景路径。

$rendersetting：当前使用的渲染设置名称。

渲染设置

$renderer：渲染器名称。如：“标准”、“物理”、或其他外部渲染器（渲染器）。

$raydepth：跟踪深度。

$refldepth：反射深度.

$shadowdepth：投影深度。

$formatdepth：输出图像的深度。如：“8-bit”或“16-bit”等。

$aafilter：渲染所使用的抗锯齿滤镜（过滤）。

$aamode：渲染所使用的抗锯齿模式。如：“几何体”或“最佳（抗锯齿）”。

$aathresh：阈值.

$aamin：最小级别。

$aamax：最大级别。

$raythresh：光线阈值。

$lod：细节级别。

$physampler：采样模式。

$phyhdrt：HDR 阈值。

全局光照

$gidiffdepth：漫射深度。

$giaccuracy：精度。

$gisamples：采样数量。

$gidensity：在下列命令中与记录密度设置结合使用：最小/最大比率，半径，最小半径，密度控制。在渲染的图像中，它会以如下方式显示：“-3/0/40%/12.5/60%”。

$gismphemi

如果设定了半球采样，则会被显示。

$gismparea

如果设定了离散面积采样、强制每像素或自定义的采样数量，则会被显示。

$gismpsky

如果设定了离散天空采样、强制每像素或自定义的采样数量，则会被显示。

$gismpradmap

如果二次反弹算法的辐射贴图选项，则会被显示。

物理渲染器

$phymblur：下列三个数值的组合（按以下顺序）：运动细分、变形细分和毛发细分。

$physampling：下列四个数值的组合（按以下顺序）：采样细分、着色细分（最小）、着色细分（最大）和着色错误阈值。

$phyblurry：下列三个数值的组合（按以下顺序）：模糊细分（最大）)，阴影细分（最大），环境吸收细分（最大）.

摄像机

随时间变化的数值（如变化的焦距数值）会在显示在对应的图像中。

$camname：活动摄像机的名称。

$camfl：焦距。

$camtdst：焦点距离。

$camwb：白平衡（K）。

$camfn：物理相机：光圈（f/#）。

$camiso：物理相机：ISO数值。

$camgain：物理相机：增益（dB）数值。

$camsht：物理相机：快门速度（秒）。

$camdstrt：物理相机：镜头畸变。

$camab：物理相机：色散。

统计

随时间变化的数值（如变化的焦距数值）会在显示在对应的图像中。

$render time：每张图像的渲染时间（如：12:34:56）。

$polycnt：渲染的工程的多边形数量。

$objcnt：工程中活动和可见的对象总数（禁用的对象和渲染不可见的对象不会计算在内；克隆对象和通过阵列复制的对象则会被包含在内）。

$lightcnt：工程中活动的光源数量（禁用的灯光和渲染不可见的灯光不会计算在内；克隆的灯光和通过阵列复制的灯光则会被包含在内）。

$starttime：渲染开始的时间。

$startdate：渲染开始的日期。

背景可见度 [0..100%]

文本框背景是由色条构成的，你可以在这里对它进行设置。如果将数值设为0%，背景会变为透明而只显示文本。

垂直对齐

可以在这里设置文本框显示在图像的顶部还是底部。

字体

可以在这里设置文本的字体。推荐使用不过于纤细的无衬线字体。可能的话，可以将字体设置为粗体，这会让文字更易读。

颜色

可以在这里设置文本颜色。

背景颜色

可以在这里设置文本框的背景色。

 图像属性

启用

如果想在渲染的图像中添加位图，可以启用这个选项。

文件名

在这里加载想要添加到渲染图像中的位图（任何 CINEMA 4D 支持的图像格式）。如果想要使用 LOGO 或类似的图形，确认它正确使用了 alpha 通道，这样可以保证只会显示 LOGO 而不会看到背景。

可见度 [0..100%]

使用这项数值来定义位图的可见度。

混合模式

参见混合模式。

水平对齐
垂直对齐

这里可以设置位图的位置：在边缘还是中心。

页边距 [0..2147483647]

如果图像不位于渲染图像的边缘或中心，可以使用这个数值来设定位图到图像边缘的距离。

缩放 [0.1..1000%]

使用这项数值来缩放位图。如果设为100%，会将位图缩放到整个表面。较低数值会减少位图的缩放比例，较高数值则会增加。

保持宽高比

启用这个选项来确保在缩放时保持宽高比。

保持原始尺寸

若启用，位图会保持其原始尺寸并按该尺寸添加到渲染的图像中。
