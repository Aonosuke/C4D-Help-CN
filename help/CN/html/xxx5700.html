Render Region

If you want to save the picture or animation, you must render to the Picture Viewer instead using the Render To Picture Viewer command. If an alert appears when you render, see Render Alerts.

Use this command to render a region in a viewport. Once you have chosen the command, drag a box in a viewport to define the region that you want to be rendered.

To access the options that control rendering, choose Render / Edit Render Settings from the main menu. The render settings are here.

To cancel rendering, press Esc or click the mouse button.

A render progress bar will appear in the bottom left corner of the screen.
