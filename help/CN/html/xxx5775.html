Load / Save Preset Tag

These options let you save the tag settings of the currently selected tag as presets. As with all presets, they will be saved to the Content Browser’s Presets / User folder, and can be deleted from this location as well.

In lieu of using the menu, Texture tags can be placed into the Presets / User folder via drag & drop. They can also be dragged from the Content Browser onto an object in the editor or in the Object Manager.
