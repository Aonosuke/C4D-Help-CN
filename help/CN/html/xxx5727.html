Lens Effects

Lens effects will be rendered if they are used in your scene. If you have edited lens effects for one or more light sources, this option can be used to enable or disable the effect for all light sources. Refer to Light Source for additional information. This effect is a typical Post Render Effect, which means it will only be visible after being rendered.
