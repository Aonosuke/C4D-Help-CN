渲染设置

通用

渲染预设选项可以用来设定图片或影片的输出格式。

这个命令会打开渲染设置对话窗，另外，这个窗口也可以通过属性管理器的模式菜单中的渲染设置选项打开。

在可用的渲染器（参见渲染器）列表的下方，你可以找到一列按类别划分的渲染选项，它们位于渲染设置窗口的左上方，后期效果或多通道效果可以通过点击相应的效果或多通道按钮来添加，多个后期效果或多通道效果可以通过拖放来改变它们的顺序（改变后期效果的顺序可以改变这些效果组合产生的结果）。如果选择了多个后期效果或多通道效果，所有相应的设置会在窗口右侧区域显示。后期效果预设可以加载或保存。

在部分渲染选项或后期效果的左边有一个复选框，你可以用它来激活或关闭相应的选项。这些复选框可以用来设定那些在渲染过程中基本不用保存的内容（无渲染的图像或视频、无多通道、无合成文件、无辐照缓存等）。

当前场景的所有渲染设置会显示在渲染设置窗口的左下角。如果选择了一项设置，它的名称和相应的图标会被高亮显示。要重命名一项渲染设置，在它的名称上双击即可（跟在对象管理器中重命名对象名称一样）。渲染设置可以直接拖放到列表中的其他位置，或按住 Ctrl 拖放来复制，甚至可以作为另一项渲染设置的子项（参见下一节）。渲染设置也可以在不同的场景之间复制和粘贴，而且它也可以被加载或保存。

渲染设置窗口的右半部分包含了全部可以被修改的渲染设置，这些设置从 R11 版本开始与属性管理器相兼容，这个版本起你可以使用所有相关的属性管理器 右键菜单命令（右键单击）和这两个额外的参数：

  • 继承父级设置：选择这个命令会重设所有没有继承值（粗体字）的参数。

  • 覆写父级设置：选择这个命令会防止任何父级数值被假定（参数名称会加粗）。

继承渲染设置

CINEMA R11 版本添加的新特性，现在可以创建继承的渲染设置。

  • 你可以直接拖放渲染设置，
    • 或者

  • 在一个渲染设置上单击右键然后选择新建子级

如果没有定义其他设置，子级渲染设置会继承所有父级的渲染设置，也就是说，如果修改了一个父级渲染设置，子级渲染设置也会相应的被修改。例外：如果一个将子级设置的参数设置为跟父级不同。

示例

继承了父级设置的子级参数名不是粗体，红色方框中的粗体参数则不会继承父级设置。

在上面的示例中，创建了一个名称为 Preview Render (large) 的渲染设置，在这个设置上单击右键并选择新建子级来创建一个子级渲染设置，将这个新的子级渲染设置重命名为 Preview Render (small)。在选项标签页中设置透明、跟踪深度、反射深度和投影深度的数值，此时这些参数会变成粗体（这表示这些数值与父级渲染设置中的不同，并且它们不会从父级渲染设置中继承数值）。

那么这样做有什么好处呢？这让你可以快速建立仅有少数参数不同的渲染预设，同时保持其他参数与父级相同。此外，改变父级设置的参数也可以相应改变它的子级设置（请记住例外情况）。

如果你改变了父级渲染设置的参数数值，那么这个参数的名称会变成粗体。这是因为这些数值与默认的数值不同，你可以通过在这个参数的右键弹出菜单中选择复位默认状态命令来去掉这个“粗体”状态。

你可以从这个列表中选择场次并做出修改，而不需要从场次管理器中来修改渲染设置（通过图标）。

继承父级行为

除非你做出了相反的改变，否则默认会使用这个模式。如果你修改了一个参数，它的名称会变为粗体。选择这个命令会将名称重设为“正常”字体。

应用 / 保存差异预设

差异预设是指只包含修改过的（粗体）参数。选择保存差异预设命令会保存该预设到内容浏览器，路径为 Presets/User/Render Settings。如果你选择了一个先前保存过的差异渲染设置，然后选择应用差异预设命令，那么只有包含在保存的预设中的那些参数会在激活的渲染设置中被修改。

加载 / 保存预设

所有渲染设置参数，包括全部后期效果和多通道都会保存到预设中。预设会被保存到内容浏览器中，路径为 Presets/User/Render Setting 。选择加载预设命令能让你从所有可用的渲染预设中进行选择。

这可以让你可以在以下任务中简单快速地保存和加载渲染预设：

  • 不需要抗锯齿、透明、投影等效果的测试渲染
  • 包含一个或多个多通道效果的最终渲染
  • 渲染一个特定范围的动画
  • 更多其他情况
