层管理器和层的概念

对象管理器的各种功能， 可以通过查看设置进行控制。除了显示对象和标签，对象管理器还提供了强大的过滤选项。

不用担心在对象管理器中找不到对象，各种过滤设置可以隐藏对象管理器中的项目。要找到某个项目，可以检查以下区域：

  • 你寻找的对象是否被过滤器隐藏了？（对象管理器：查看 | 使用过滤）
  • 是否在搜索条中输入了文字来过滤项目？（对象管理器：查看 | 显示搜索条）
  • 丢失的项目是否在隐藏的层？（层管理器：查看 | 管理器）

对象管理器是所有对象及其标签的控制中心。你可以自由选择哪些物体要显示在对象管理器中。你在建模所以只想显示多边形对象？没问题，选择要显示的这个对象，然后隐藏其他所有对象。你的场景非常复杂而现在只想编辑其中的 17 个灯光？没问题，将其他对象都隐藏起来！

立方体的绿色层被隐藏之前（图左）和被隐藏之后（图右）。

虚线表示真实层级关系中有一部分被隐藏了（实际是存在的并可以被渲染）。

这些特性赋予了对象管理器极大的灵活性，根据查看模式和过滤器的不同，对象管理器显示的内容会完全不同。强大的过滤器让你可以完全控制哪些项目要显示在对象管理器中。你最多可以打开四个对象管理器，并将它们集成在界面中。

无论场景多复杂，你可以轻易地进行管理。书签让你可以快速在不同地查看选项之间切换。

对象管理器的主要任务：

  • 管理对象的名称、层级、可见性等等。
  • 管理其他项目，例如你可以在这里给对象添加标签。
  • 对象管理器搭配层管理器一起使用，你可以将不同的项目分配到不同的层，并通过层来进行管理。例如，你可以隐藏或锁定特定层的所有项目。

对象管理器会使用下面的默认设置，这时搜索条、路径条和过滤器都是隐藏的。

对象管理器左侧包含了场景中的所有对象，以树状结构显示。

要展开或折叠分支，可以执行下面操作其中之一：

  • 点击对象名称旁的+或-按钮。按住Ctrl/Cmd点击则会完全展开或折叠整个分支。
  • 按左或右方向键可以展开或折叠当前活动的分支。

你可以通过拖放重新排列对象管理器中的项目或群组。按住Ctrl/Cmd拖放则会复制项目。你还可以将对象设置为其他对象的子对象。

使用过滤器和搜索条可以控制哪些对象可以显示。关于对象、标签和材质的描述，请查看帮助文档的相关章节。

通过点击、框选或使用方向键（如果已经选中了项目）来选择项目。同时按住Ctrl/Cmd可以进行多选，按住Shift则可以连选。

在层级中可以使用下列快捷键：

  • 中键点击：选择点击的对象及其所有子对象。
  • Alt + 中键点击：选择点击的对象和跟它位于同一层级的对象（不包括子对象）。
    • 进行多选的方式主要有：

  • 画选框。
  • 点击一个项目然后按住 Shift 点击另一个项目，这时这两个项目之间的其他项目都会被选中。
  • 按住 Shift 的同时按上或下方向键。
  • 使用对象管理器编辑菜单中的选择命令。
  • 按住Ctrl/Cmd键点击来选择/取消选择对象。

如果选择了多个项目，属性管理器会显示它们共同的设置，你可以对其进行编辑。如果只选择了一个项目，则会显示它所有的设置。

对象管理器中被选择的对象会高光显示。

RELEASE 17

最后一个被选中对象的名称的颜色会比其他的更亮一些（参见最上面图片的右上部分）。这个特点对于像样条布尔命令这样的功能会有帮助。

所选对象的子对象也会高光显示，但是如果子对象本身没有被选中的话，它们的颜色会稍微暗淡一些。

要重命名一个对象，你可以：

  • 双击它们的名称。
  • 选择一个对象然后按回车键，会出现对象名输入框。你可以使用上下光标移动到下一个对象上来快速重命名多个对象，这时就不需要按回车键了。
    • 对象管理器中间一列包含了各种开关，各种对象所能使用的开关有所不同，从左到右分别是：

    层颜色、编辑器/渲染器 开/关（你也可以在基本属性标签页或对象管理器的对象|对象显示子菜单中找到这些开关）、对象激活（开/关），如果是摄像头，则会显示一个以摄像机的透视视角查看的开关。

你可以对层、编辑器/渲染器 开/关和对象激活使用笔刷来批量切换开关状态（按住左键划动），这时鼠标会变成一个笔刷。只需要在开关上向上或向下滑动即可。

对象管理器的右边显示独享的标签。

CINEMA 4D 中的对象总是有一个特殊的类型，如多边形对象、灯光或标签。点击类型图标会在属性管理器中显示所有属性（）。 An object in Cinema 4D always has a particular type, such as polygon object, light source or tag. Click on the type icon to display all 属性 in the 属性管理器 (object-by-object descriptions of these properties together with the type icons).

拖放

拖放 is the technique of clicking on an object, holding down the mouse button and dragging the object to another position; when the target location is reached, you release the mouse button and the object will be dropped there. Depending on the target location, this may have different results.

重新排列对象

There are many ways to re-arrange objects in the 对象管理器 and these are described below. The icons show how the mouse pointer changes while doing the actions as described.

Drag an object between two others or to the end of the list.

If you want drag to create a copy, use Ctrl-drag.

改变对象层级

To change the object hierarchy, drag the object on top of an existing one in the list. This makes the dragged object a child of the other.

If you want to create a copy and make it a child of another object, use Ctrl-drag and move the mouse pointer over an object.

You can also drag-and-drop tags. To transfer a tag from one object to another, drag the tag icon on to the line of the other object.
