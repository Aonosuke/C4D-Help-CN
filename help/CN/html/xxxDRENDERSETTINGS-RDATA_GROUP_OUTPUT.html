输出

All Output settings only affect rendering in separate editor windows. Since only single images or a part of an image can be rendered in the Viewport, these settings do not affect this rendering.

预置

点击这个按钮会打开所有常用的预置分辨率，其范围囊括屏幕、出版打印、视频和胶片。

这些预置会自动设置分辨率、宽高比和帧频，你需要做的只是调整窗口下方的五个参数，用它们可以调整视频长度。

注意，屏幕/编辑器分辨率选项会按你的视窗的分辨率进行渲染。如果改变了视窗的尺寸，那么需要重新打开这个预置并再次进行设置。

示例：

假设你想要创建一个 iPod 视频，只需要简单的 3 步：1）点击预置按钮然后选择 互联网 | iPod Video 640×480 预置；2）切换到“保存”页面然后从格式菜单中选择 QuickTime 影片；3）点击格式菜单右侧的选项按钮，然后选择 H.264 或者 MPEG-4 Vedio 压缩设置。现在你就可以准备创建你的 iPod 视频了。

你甚至可以为 iPod 视频添加一个声音文件（参见声音），你需要在保存页面启用包括声音选项。

宽度 [0..128000]
高度 [0..128000]

这两个参数用来设置图片的输出高度和宽度。选择一个预置分辨率会自动生成相应的宽度和高度值。如果需要的话，宽度数值框右侧的下拉菜单可以切换其他单位。无论你设置成什么单位，CINEMA 4D 都会按照设定的像素数（显示在分辨率设置下面）来创建位图和视频。对于像素外的其他单位，其数值会根据分辨率设置进行计算。

你可以问一个问题来决定应该使用哪个分辨率：“这个图片/视频是用在什么地方的？”

一旦决定了，你就可以从预置菜单中选择合适的预置，用低分辨率进行测试渲染来加快工作流程。

提示：
宽度和高度各自的上限为 16000（32位）和 128000（64位）像素。不过几乎没有图片或图片会用到这么极端的分辨率。格式相关的更多信息可以参考这里。

锁定比率

当启用了这个选项时，修改宽度或高度其中的一项，另一项会相应地调整，其调整依据是胶片宽高比设置。


分辨率 [1..100000]

输入一个 DPI 数值，印刷图片会使用该分辨率（不用应用到视频上！）。分辨率数值定义了图片会打印为何种尺寸。因为大部分 CINEMA 4D 生成的图片都会用其他图片编辑软件或输出软件进行修改，因此你不需要修改这里的设置。

示例：

如果一个图片按 700x1000 像素的分辨率来渲染，并保存为 72 DPI，那么其整体尺寸为 24.7x35.3 厘米。如果同样的图片保存为 300 DPI，那么其整体尺寸为 5.9x8.5 厘米。

渲染区域

左图：禁用区域渲染；右图：启用区域渲染。

你一定对这种情况很熟悉：你刚刚渲染了一个复杂的场景，现在要做一点修改，它只会对整个画面中的一个小区域产生影响。通常一旦做出了修改，你就需要再次渲染整个场景。

渲染区域功能可以让你不需要只因为一点小修改就得渲染整个场景，只有选择的区域会被渲染，而剩下的图片会显示为黑色，这样可以大大加速你的工作流程。渲染的区域可以之后复制到先前的渲染图片中。

左侧边框 [0..128000]
顶部边框 [0..128000]
右侧边框 [0..128000]
底部边框 [0..128000]

使用这两个数值可以设定想要渲染的区域边界（单位为像素）。注意，如果输入了无效的数值，则不会创建边界（例如输入的数值超过整个画面的尺寸）。

复制自交互区域渲染(IRR)

不用凭感觉使用上面的设置来定义边界，交互式区域渲染可以在视窗中创建，然后使用复制自交互区域渲染(IRR)选项将其边界坐标复制到上述的数值框中。

胶片宽高比 [0.001..1000]

该参数表示渲染区域的宽度和高度比。选择自定义选项来手动输入数值。这个下拉菜单也包含了一些预置的选项。

高度和胶片宽高比的数值是直接相关的，如果修改了宽高比，则高度的数值也会相应地改变。

示例：

设置宽度为 320 像素、高度为 240 像素，这时宽高比为 4:3（即 1.333:1）。选择胶片宽高比的标准(4:3)选项会自动进行设定。现在把胶片宽高比选项改为 HDTV 16:9，CINEMA 4D 会自动修改图片的分辨率为 320×180 像素。

同样的图片分辨率也可以在视窗中进行设置。在视窗的顶部、底部、左侧、右侧你都会看到灰色的边框，它们是渲染安全框，位于其中的区域会出现在渲染文件中。

像素宽高比 [0.001..1000]

像素宽高比对许多输出设备都很重要，它定义了一个像素在屏幕上的宽度和高度。对大多数屏幕来说其像素宽高比都是 1:1（正方）。

但是有些显示媒介使用不同于 1:1 的像素宽高比，所以必须调整设置避免诸如圆形显示为椭圆这种情况。

如果你需要手动计算像素宽高比，你可以拉伸侧视图到整个屏幕，然后创建一个立方体。用尺子测量立方体的宽度和高度，并将它们分别输入到两个数值框中获取像素宽高比。

帧频 [1..2500]

设置渲染的帧频（参见配置 CINEMA 4D）。这里的设置独立于工程设置中定义的帧频。

帧范围

手动

如果你想要渲染帧序列，但是不需要整个动画，可以在上面的输入框中输入起点帧，在下面的输入框中输入终点帧。帧范围的下拉列表会自动变为手动。如果你想要保存这个动画，请在保存选项页输入保存路径。

如果你停止了 AVI 或 QuickTime 影片的渲染，所有的帧都会被丢失，不能从停止处继续渲染。你可以渲染一些短的视频序列，然后在合适的视频编辑器中进行合成。

当前帧

只有当前帧会被渲染，如果你想要在渲染后自动保存动画，请在保存选项页输入保存路径。

全部帧

无论是渲染为图片序列还是渲染为影片（AVI、QuickTime），所有帧都会被渲染。如果你想要渲染后自动保存动画，请在保存选项页输入保存路径。

如果你停止了 AVI 或 QuickTime 影片的渲染，所有的帧都会被丢失，不能从停止处继续渲染。你可以渲染一些短的视频序列，然后在合适的视频编辑器中进行合成。

预览范围

箭头标记出了预览范围

只有预览范围（时间线中位于两个标记之间的区域）会被渲染，无论是渲染为图片序列还是影片（AVI、QuickTime）。如果你想要渲染后自动保存动画，请在保存选项页输入保存路径。

如果你停止了 AVI 或 QuickTime 影片的渲染，所有的帧都会被丢失，不能从停止处继续渲染。你可以渲染一些短的视频序列，然后在合适的视频编辑器中进行合成。

起点
终点

如果你设置这个选项为手动，那么你可以手动输入要渲染的帧范围。

帧步幅 [1..1000000]

使用这个设置可以定义是否每个动画帧都要被输入，还是每第二帧、第三帧、第四帧等被输出。假定你有一个包含1500帧的动画，现在你要做一个测试渲染。如果帧步幅设置为10，那么每10帧会渲染1帧，渲染的结果会看起来只是一个粗略的印象。

场

当渲染视频时，用场渲染来获取平滑的播放效果。连续的场以半个帧时长相互分离。渲染器会创建交错的输出帧，这时每个帧都包含两个场。对每个输出帧来说，在第一个场的半个帧时长后会渲染第二个场。这两个场是交错的，其中一个场包含奇数线（奇数场），另一个场包含偶数线（偶数场）。

PAL 制式的视频（用于欧洲等地）每秒为25帧，因此每秒包含50个场，NTSC 制式的视频（用于北美、日本等地）每秒为30帧，因此每秒包含60个场。

不要在静帧中使用场渲染，它只适用于视频输出。

当渲染场时，使用无损压缩或不压缩。有损压缩（例如，JPEG、M-JPEG）会让场模糊，它们不适合场渲染，使用了这种压缩器的渲染结果会很糟糕。

避免将场渲染与景深效果（CINEMA 4D Visualize 和 Studio）一起使用，否则会严重影响图像质量。

无

只有完整的帧会被渲染。如果你要渲染静帧或不在视频系统中观看的影片，请使用这个设置。

偶数优先

一个动画图像的偶数场会与另一个图像的偶数场混合。

奇数优先

一个动画图像的奇数场会与另一个图像的奇数场混合。

提示：
不推荐与景深效果一同使用场渲染，图像质量会严重受损。
