背景

你可以在平面视图中放置一张背景图片来来辅助建模。例如你在建一辆汽车模型时，你可以加载一定比例的汽车示意图到顶视图、右视图和前视图，然后利用这些示意图帮助你建立精确的汽车模型。当你缩放视图时图片也会自动进行缩放。

图片

点击三个点的按钮然后在打开的文件选择窗口中选取背景图片。任何 CINEMA 4D 能识别的位图格式都能被加载。

你可以直接从你的电脑中直接拖放一张图片到视窗中来放置背景。

显示图片

你可以在这里暂时切换显示或关闭背景图片。

模式

这些模式定义了当图片放大时，加载到背景中图片的像素会如何生成新的像素。邻近产生锐利的边缘，而线性会产生模糊的。

保持比例

确认背景图片在缩放时会保持它的原始比例。

水平偏移 [-∞..+∞]
垂直偏移 [-∞..+∞]
水平尺寸 [-∞..+∞]
垂直尺寸 [-∞..+∞]

你可以在水平或垂直方向上移动或缩放背景图片。

旋转 [-∞..+∞°]

这个参数允许你旋转背景图片。

透明 [0..99%]

这个参数定义背景图片的透明度。

Alpha 模式

如果背景图片有一个 alpha 通道，你可以使用这个设置指定这个 alpha 通道应该如何表现。

无

忽略 alpha 通道。

正常

使用 alpha 通道遮挡（隐藏）部分图片。

反转

与正常模式相同，只是显示的部分与隐藏的部分反转。

网格

这里可以设置网格的属性。

传统模式

这个选项你可以切换网格的表现。如果勾选，会在所有视图中使用自动网格，同时参考线在缩放视图时会自动显示或隐藏。

网格间隔 [0..+∞m]

网格间距是两条网格线之间的距离。

线条 [0..2147483647]

线条选项可以让你选择网格线的总数（拉伸 X 轴和 Z 轴），默认值是100。当你选择一个更高的值时，网格会在水平面上延伸。这个设置只关系到透视视图。

主线周期 [1..2147483647]

网格包含以暗色凸显的主间隔线，通过这个选项你可以选择这些间隔线出现的频率。

动态网格

用这个选项你可以总是在屏幕上获得一个合适的网格宽度，与选择的网格间距无关。它只应用于 2D 视窗。使用下拉菜单来选择动态网格的参数。

无

如果你选择无，网格不会动态显示，网格间距（使用默认值100）会保持固定，无论你怎样缩放。

1..10

如果你选择1...10，每缩放10步网格会进行适应。如果放大到对视图来说空间太宽，CINEMA 4D 会自动切换到网格间距为10。如果继续放大，间距会变成1。如果从默认网格缩小，100的网格宽度太过密集无法清晰显示，它会先切换网格间距为1000，如果继续缩小则会切换到更高的网格间距。间距没有上下限制。

1..5..10
1..2..5..10
1..2,5..5..10

选择 1...5...10 表示网格宽度可以是 1, 5, 10, 50, 100, 500 等。但如果选择1...2...5...10 则表示如网格宽度可以是 0.1, 0.2, 0.5, 1, 2, 5, 10, 20, 50等。
