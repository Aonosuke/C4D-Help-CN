HUD

你可以用参数名称旁边的圆框直接在 HUD 中记录关键帧，还可以拖动帧元素（右下角）来前进或后退动画。

HUD（Head-Up Display）来源于航空，在飞行器中，HUD 指投射到挡风玻璃上的读数，让飞行员可以不用低头就可以看到这些信息。

HUD 在 CINEMA 4D 中的作用类似，直接在视图中显示参数，在这里你还可以编辑或制作动画。

HUD 可以显示四种类型的信息：

  • 对象参数。你可以把属性管理器中的任何对象参数添加到 HUD 中显示，并且可以用实时选项回馈（拖动数值）直接在 HUD 中直接调整它们，你还可以直接在 HUD 中记录帧。
  • 一般信息。你可以让 HUD 显示各种一般信息，例如活动对象离摄像机有多远，选中点的数量等等。

  • 对象可以直接从对象管理器拖放到编辑器视图中，此时会创建一个包含所有图标的 HUD 元素来帮助选择相应的元素。
  • 命令可以直接从自定义命令管理器拖到 HUD 中。

你可以在视窗菜单的过滤器页面开关 HUD。要打开视窗设置，在视图菜单中选择选项 / 配置视图。

HUD 元素可以包含在渲染的图片和动画中 — 在渲染设置的选项页面，启用渲染 HUD。

使用 HUD

对象参数

要添加对象参数到 HUD：

  • 在属性管理器中选择参数。（要选择所有参数，可以在上下文菜单中选择全部选择。）
  • 在参数上点击右键并在上下文菜单中选择添加到 HUD。
  • 单个参数也可以直接拖放到 HUD 中。

你可以直接在 HUD 中调整参数，点击并拖动一个参数来调整其数值。如果参数有下拉菜单，点击参数来打开其菜单。有两个状态（开启和关闭）的选项会在开启时显示为黄色，点击选项可以切换状态。

你还可以在 HUD 中直接记录动画帧。这跟在属性管理器中记录参数一样：要记录一个帧，在参数上按Ctrl/Cmd+左键。再次按住Shift+click 则会移除这个关键帧。在参数的圆圈上按住Ctrl/Cmd+Shift+左键可以删除参数的轨迹，包括所有分配给它的关键帧。

如果你想的话，你可在视图中多个地方显示同一个参数。

在 HUD 中可以使用如下快捷键：

  • Shift+左键多选 HUD 元素。选中的 HUD 元素会标记黄色边框。
  • 在 HUD 元素（名称或数值）上双击可以打开一个文本框，你可以为这个元素输入新的名称或数值。
  • 按住 Ctrl 可以拖动 HUD 元素。

选择助手

当从对象管理器拖放一个对象到编辑器视图中时会自动创建一个 HUD 元素。这是一个选择助手功能，并使得一个可选择的 4D 命令可用（在Action HUD 元素上单击右键）。在 HUD 元素上点击可以选择相应的对象或命令。

这对角色动画尤为有效，因为大部分可视化选择器功能能集成到 HUD 中。

提示：
双击选集可以选择所有的子对象。

一般信息

视窗设置中的 HUD 标签允许你选择在 HUD 中显示哪种信息类型。要打开视窗设置，在视窗菜单中选择选项 / 配置视图。

你可以直接在视图中直接改变某些一般信息的参数。要实现这一点，可以在参数上双击并在打开的文本框中输入新的数值。

HUD 设置

在一个 HUD 元素上单击右键可以打开如下设置（出现的设置会根据点击的元素类型的不同而有差异）：

显示

这些设置在主要控制一个对象的 HUD 元素什么时候显示：

同类型对象激活：当同类型的任意对象激活时。例如，一个灯光的 HUD 会在你激活场景中任意灯光时显示。

这允许你用一个 HUD 元素调整多个对象的参数（如多个灯光的亮度）。

父对象激活：当层级链中任意一个更高层级的对象被激活。

同级对象激活：当层级链中任意对象被激活。

总是：一直显示。

参数激活，设置参数：这两项设置可以用属性管理器中的复选框（见下面的数值框）来显示或隐藏单独的 HUD 元素，但仅应用于相应的对象、标签或工具，也就是说你不能使用一个球体的 HUD 设置来影响一个立方体。为某个 HUD 元素选择设置参数命令并在文本框中输入明确的选项名称（确认大小写匹配）。

指示

对象：在每个参数前面显示对象名称

名称：显示参数名称（源对象名称+你输入的）。

边框：在参数群组周围显示一个边框。

关键帧：显示圆圈，让你可以为参数记录动画关键帧。

图标：隐藏或显示一个参数或 HUD 元素图标。

自动折叠：

如果启用了这个选项（每个 HUD 元素需要单独设置），把鼠标移到元素名称右侧的小三角时元素会被折叠（关闭）或展开（全部显示），当元素被取消选择时它会自动隐藏。

颜色：这个选项包含一个为元素添加颜色的菜单，单独的颜色可以分配给文本、背景和框架。复位默认状态选项会起到明显作用。

打开图标：当一个对象直接从对象管理器拖放到编辑器视图中时会显示一个 HUD 元素，同时会显示对象的图标。这个对象图标可以用你自定义的图标替换，所有 CINEMA 4D 能读取的图像格式都可以使用，图像中的任何 alpha 通道都会被应用。

确认图像路径是与场景文件相关的，这表示图像应该放置到各自的场景目录下。

图标尺寸你可以从下面选择：小、中等、大、原始（与原始图像相同尺寸）、适合（适合文本高度）。

动作

下列命令只对上述的选择助手有效（从对象管理器中拖放到编辑器视图中的对象也是），使用这个选项可以定义 CINEMA 4D 被执行的程序或者当选项被选择时会切换到哪种模式。

启用：定义当点击 HUD 元素时是否执行一个命令。如果这个选项没有启用，则只会选择相应的对象。

符号

如果一个 HUD 元素被 2D Vector Field 定义，那么 Symbol 菜单可用，你可以用它选择小十字的造型。

滑块

这些选项控制滑块的外观。

布局

你可以在这里找到在参数群组内控制多个选择的参数排布的选项。

删除群组

从一个 HUD 群组中删除选择的参数，这个参数会成为一个单独的 HUD 元素。

删除

从 HUD 中删除选择的对象。

显示属性

在属性管理器中显示参数所属对象的属性。

复位默认状态

将选择的（也可以多选）的参数复位到对象初始的状态，也就是对象被创建时的状态。

其他

不在参数群组中的单个参数，有以下三个选项可用：

绝对：如果你已经添加了一个参数群组到 HUD 中，这个选项会决定在缩放时，参数之间的距离是保持相同（禁用选项）还是按比例变化。

重命名：如果 显示 | 名称 选项启用，可以在这里改变 HUD 名称（添加到对象名称中）。

跟随对象：将 HUD 对象锁定到对象相对位置上，即 HUD 元素会随对象一起移动。

垂直：将 HUD 元素逆时针旋转90°。

简述如何创建一个包含动画工具栏播放选项的 HUD 命令群组：

  • 打开自定义命令管理器并启用编辑面板图标选项，从面板中拖动图标、或从自定义命令管理器中拖动命令到编辑器视图（后者必须禁用编辑面板图标命令）。
  • 禁用编辑面板图标命令并用 Shift 键选择所有 HUD 命令，一旦全部被选中，在其中一个上面单机右键并选择创建群组。
  • 如果只有图标被显示，重新选择所有图标并单击右键。选择显示 / 对象选项，现在只有每个图标各自的图标被显示，可以按住 Ctrl 键移动这些图标到你想要的地方。同样的菜单还有移除群组名称和开关群组箭头等的选项。

刷新：如果你发现属性管理器和 HUD 之间的显示出现差别，可以使用这个选项。

重命名：如果 显示 | 名称 选项启用，HUD 名称会改变（原始对象名称+你输入的）。

创建群组：在编辑器视图中可以选择几个命令创建一个群组（从命令管理器或对象管理器中拖到编辑器视图中命令）。
