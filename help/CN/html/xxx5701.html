Render To Picture Viewer

This command renders the scene to the Picture Viewer. Once rendering starts, a progress bar appears in the status bar. The bar shows the time elapsed since the render started. If you are rendering an animation, the bar also shows you the current frame number and the total number of frames.

To access the options that control rendering, choose Render | Render Settings from the main menu.

To cancel the rendering, press Esc.

If you want to save the picture or animation, enter a filename in the render settings before you use this function.

Attention!

Images and animations can only be saved when rendered in the Picture Viewer. Save Path, File Name and corresponding options must be defined in the Render Settings … menu (images not saved in the Picture Viewer can still be subsequently saved).
