Hardware Preview

Selecting the Hardware OpenGL option in the Basic Properties menu’s Renderer menu will cause render output to result in Hardware OpenGL (see Hardware OpenGL aktivieren) format (a very concise render output compared to the comprehensive output but therefore very fast thanks to graphics card acceleration).
