Software Preview

If you select Software OpenGL from the Render Settings’ Renderer menu, rendering will be done using Software OpenGL (see also Activate Hardware OpenGL). The quality is quite modest compared to the normal render quality but is calculated very quickly (but much slower than with Hardware OpenGL). No graphics card acceleration at all is used.
