色彩校正

色彩校正

饱和度[-100..100%]

决定了图像中颜色的饱和度，即颜色的鲜艳程度。饱和度为 0 时图像会变成灰度图。

亮度[-100..100%]

让图像均匀地变亮或变暗。

对比度 [-100..100%]

对比度越高，画面中的暗部会越暗，而亮部会越亮，这会扩大画面中不同亮度的区域的差别。

对比度越低，画面中的暗部会越亮，而亮部会越暗，这回让图像看上去变“平”。

曝光 [-10..10]

曝光可以调节 HDR 和正常图像的亮度。

Gamma [0.1..10]

Gamma 值决定了亮度值会如何显示（如某一亮度的像素会变得很亮或很暗）。Gamma 值基本是用来调节亮度的。

红
绿
蓝

使用这些选项可以开启或禁用各个颜色通道。

反向

反转颜色的色值，黑色会变成白色，反之亦然。RGB 图像的的反色是，对于每个像素的每个通道，用颜色通道的最大值（例如8位图像是255）减去像素的色值，结果就是该通道的反色色值。

暗调值 [0..100]
明度值 [0..100]

这两个数值决定了最亮点和最暗点。如果改变了其中一个数值，图像的色调也会相应进行缩放。

渐变曲线

渐变曲线可以提亮、加深或着色指定的亮度区域。

渐变曲线可以用来精确地修改原始图像的亮度和颜色区域。例如只提亮图像中的暗部或给亮部调色。

用单独的红、绿、蓝色渐变曲线还能对单个通道进行调节。

在图片查看器的滤镜设置中也能找到颜色校正设置，可以用它来交互式地调节图像。

分级强度 [0..100%]

这项设置可以用来平滑地调节色彩校正的强度。设为 0% 表示禁用该效果，设为 100% 则是将效果最大化。

曲线最小 [-∞..+∞]
曲线最大 [-∞..+∞]

这两项设置可以定义曲线生效的亮度区域。对于是8位（0-255）还是16位（0-65535）并不重要，默认的数值是 0 和 1。

但是这两个设置与使用 HDR 图像有关。HDR 图像可以包含几乎无限的亮度，使用曲线最小和曲线最大值可以更加精确地限制曲线生效的亮度范围。低于曲线最小以及大于曲线最大（可以输入超过1000的数值）的亮度值会保持不变（注意：这意味着如果使用了默认的曲线最大数值 1，所有的亮度区域都会被忽略。因此曲线最大的数值需要手动调整），渐变曲线会在数值范围内生效。

重置参数

点击这个按钮会将所有色彩校正数值恢复为初始值，不再有色彩校正效果。
