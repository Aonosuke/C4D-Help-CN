Flush Illumination Caches

If you enable Save Solution on the Global Illumination or Caustics pages, files will be saved in the scene’s Illum folder so that subsequent renders can reuse the data. Choose Flush Illumination Caches to delete these files. The command will also flush cached shadow maps (render settings, Options page).
