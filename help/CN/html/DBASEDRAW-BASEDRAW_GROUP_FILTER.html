Filter

Almost any type of element that can be displayed in the viewport can be switched on or off here on the Filter tab. The options are self-explanatory except for the following:

Environment

This refers to environment objects such as the Floor, Background, etc.

Other

This refers to elements that do not fall into any other category such as loudspeakers, etc.

Multi-Select Axes

When multiple objects are selected a small object axis will be displayed for each. Use this option to turn the display of these axes on or off.

Axis Bands

This refers to the axis bands that appear in the viewport when the Move tool or Scale tool is selected (these bands enable you to move or scale along a plane).

N-gon Lines

N-Gon Lines disabled (left) and enabled (right).
N-Gon Lines disabled (left) and enabled (right).

 

Each n-gon will be internally triangulated for rendering or animation. You can display these triangles in the viewport by enabling this option.

Ghosting

Disables the Display tag’s integrated Ghosting functionality.

Guides

Use this setting to define the visibility of the guidelines (as an object or dynamically).

Gradient

This setting can be used make a monotone surface out of the background’s grayscale gradient (which will not be rendered) displayed in the Viewport.

Navigation Cross

Enable this option to display the target or rotational center point as a Navigation Cross in the Viewport when it is zoomed or rotated.

Null

Enables/Disables the display of Null objects.

Polygon

Enables/Disables the display of polygons.

Spline

Enables/Disables the display of Splines.

Generator

Enables/Disables the display of Generators.

Subdivision Surface

Enables/Disables the display of Subdivision Surfaces.

Deformer

Enables/Disables the display of Deformers.

Camera

Enables/Disables the display of cameras.

Light

Enables/Disables the display of lights.

Particle

Enables/Disables the display of Particles.

Joint

Enables/Disables the display of Joints.

Grid

Enables/Disables the grid display.

Horizon

Enables/Disables the horizon display.

World Axis

Enables/Disables the World Axis display.

Bounding-Box

Enables/Disables the bounding box display.

HUD

Enables/Disables the HUD display.

SDS Mesh

Enables/Disables the Subdivision Surfaces mesh display.

Component Highlighting

Elements that can be selected (points, edges, polygons) will be highlighted when the cursor lies over them.

Object Highlighting

Enable this option of objects should be highlighted with an outline when the mouse approaches.

Highlight Handles

Enables/Disables the Highlight Handles function.

Axis

Enables/Disables the display of axes.

SDS Cage

Deactivates the display of the unsmoothed Subdivision Surfaces Cage object in point, edge or polygon mode.
