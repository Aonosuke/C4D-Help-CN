HUD

 信息

这里你可以选择在 HUD 中显示哪种类型的信息。

摄像机距离

显示摄像机与焦点（Point of Interest）之间的距离（参见摄像机模式）。

帧

显示当前帧。

帧时间

显示当前时间。

摄像机

显示当前摄像机。

投射

显示投射类型。

对象名称

显示激活对象的名称。

根对象

当一个层级中的对象被选中时，显示这个层级中最顶部对象的名称。

父对象

当一个层级中的对象被选中时，显示这个对象的父对象的名称。

总计对象

显示当前场景中对象的总数。

激活对象

显示选择对象的总数。

总计点

显示所选对象点的总数。

所选取点

显示选择的点的数量。

总计边

显示所选对象边的总数。

所选取边

显示选择的点的数量。

总计多边形

显示所选对象多边形的总数。

选取多边形

显示选择的多边形的数量。

总计 N-gons

显示所选对象 N-gons 的总数。

选取 N-gons

显示选择的 N-gons 的数量。

帧率(FPS)

显示帧率。

统计学

勾选这个选项会在 HUD 中显示一些 OpenGL 统计数据（如三角形、点的数量等，也包括网格、高光元素、灯光、摄像机等。你不会看到场景中几何体自身的信息，但是会看到用 OpenGL 显示的元素的信息。）。这些统计数据只作为信息显示，没有其他功能。关闭这个选项会在 HUD 中移除这些信息的显示。

工具

显示一个包含最近使用的8个工具的小菜单，用来快速选择工具。

雕刻统计

在雕刻时，相关的统计数据（如多边形数据）会显示在视窗右上角的 HUD 中。使用这个选项来打开或关闭显示。

 控制

 

这些设置让你可以调整 HUD 的视觉外观（它的颜色和透明度）。

背景

用来设置 HUD 的背景颜色。

透明 [0..100%]

用来设置 HUD 的背景颜色的透明度。

文本

用来设置文本颜色。

透明 [0..100%]

用来设置文本的透明度。

选取对象

用来设置激活的 HUD 元素的文本颜色。

始终激活

如果勾选了这个选项，你将不能控制位于 HUD 下方的元素，鼠标动作只能影响到 HUD 元素。

我们建议你禁用这个选项，因为这样 HUD 元素和位于 HUD 下方的对象都能被编辑，步骤如下：We recommend you deactivate this setting since both HUD elements and objects lying beneath the HUD can be edited. The following applies:

  • 点击 + ALT键：鼠标操作 HUD

  • 不按额外的按键：HUD 会被忽略，鼠标只能操作位于 HUD 下方的对象。
