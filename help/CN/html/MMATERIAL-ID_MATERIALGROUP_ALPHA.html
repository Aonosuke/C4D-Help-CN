Alpha

左图是将一个树叶贴图投射到各个略不相同的表面上。右图使用了 Alpha 通道，它“裁剪”出了叶片表面。场景版权 © Milan Soukup

Alpha 通道让你可以使用一张图像来遮挡材质上的区域，让背景显露出来。它对于细节的提升十分有用。它的原理是定义材质的一些区域让其不可见，这样位于这些区域后方的材质或对象会显露出来。3D 树木就是一个经典的示例。比起给单独的树木进行建模，还可以把树木的图像添加到对象上，然后用 Alpha 通道剪出树木的轮廓。

许多 CINEMA 4D 的着色器有内置的 Alpha 通道。

如果在渲染带有多个 Alpha 通道的对象时出现了黑色区域，可以在渲染设置的选项页面增加跟踪深度

如果将一个带有 Alpha 通道的材质应用到一个没有其他底层材质的对象上，那么该对象在 Alpha 通道允许显示底层材质的区域不可见。

假设你想要讲一个扫描的树木放到场景中。首先，你需要创建一个材质，启用颜色通道，选择一张树木图片作为颜色贴图，然后将这个材质分配一个对象，如矩形多边形。

当渲染这个场景时，你会在这个矩形多边形上看到树木，但是树木周围的区域是不透明的，而是有一个蓝天背景（图 1）。

图 1 。

现在编辑树木材质，启用 Alpha 通道，切换到 Alpha 通道并加载树木贴图。先禁用柔和选项，然后按住 Ctrl 点击想要去掉区域的颜色——在这个示例中，是树木周围的天空（图 2）。

图 2 。

CINEMA 4D 会将颜色设置为这个颜色，并根据这个颜色来遮盖图片。现在这棵树周围没有天空背景了（前提是启用了光影着色或快速着色）。现在你可以在 Alpha 页面选择一个稍微不同的颜色，然后可以立即看到效果的变化。不过在叶片、树枝和树干周围仍然会残留一些背景，因此现在的结果还不完善。

要得到更精确的结果，可以在图像编辑软件中创建 Alpha 通道然后添加到树木图像上。将这个图像加载到 Alpha 通道然后启用柔和和图像 Alpha 选项。现在贴图的遮罩就很干净了（图 3）。

Alpha 通道和抗锯齿过滤器

Sinc 抗锯齿过滤器会扩大边缘。

根据抗锯齿中的过滤设置，Alpha 通道的边缘可以可以进行平滑。这可能会在接下来的合成中出现问题，尤其是当使用 Sinc 过滤时，会出现如上图的问题。根据设置，抗锯齿过滤器的效果会超过对象边缘，这个问题可以通过选择其他过滤器或使用较小的滤镜宽度和滤镜高度来减少。

颜色
Alpha 容差（变量）

如果对使用了抗锯齿的贴图进行裁剪，会在对象周围产生一个明亮的边界，这是由主贴图和 Alpha 颜色之间的颜色抗锯齿造成的。这个问题可以通过调节这两个滑块解决。

你可以使用颜色对图像中特定区域进行裁剪。使用这种方式，你需要从选择的图像中选择想要变成透明的颜色。缺点是你想要变透明的区域通常包含几种颜色，因此会在物体周围留下一圈杂色。但是通过设置Alpha 容差（变量）颜色，你可以设置颜色偏离的程度，这有助于减少杂色残留。

使用贴图内含的 Alpha 通道可以清晰精确地裁剪贴图（参见下面的图像 Alpha）。

反向

这个选项会反向 alpha 通道，透明部分和实体部分将反转，这对颜色裁剪和贴图内置 Alpha 通道两种方式都有效。

柔和

启用这个选项会让贴图边缘渐渐消失，这会让你有更多方法创建真是视觉的对象。柔和默认是启用的，这时颜色和Alpha 容差（变量）是无效的。纹理贴图现在的作用是决定渐隐的范围。贴图中白色的像素表示该处的材质是100%不透明的。如果贴图是黑色的，则是100%透明的。

图像 Alpha

启用图像 Alpha时，你可以使用加载的图像中已有的 Alpha 通道。支持 TIF、TGA、PICT、Photoshop PSD 和 QuickTime MOV 图像格式（所有 QuickTime 支持的格式）。如果图像中没有 Alpha 通道，那么图像 Alpha 通道选项会被忽略。可以使用反向功能来反转 Alpha 通道，而不需要在图像编辑软件中重新制作贴图。

预乘

如果使用带有预乘 Alpha 通道，可以启用这个选项。要确认图像软件生成的是这种类型的 Alpha 通道。

纹理

可以在这里设置一张纹理贴图或一个2D着色器。详情参见纹理。
